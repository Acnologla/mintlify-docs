# Como Jogar

Bem-vindo ao **Asura Bot** — o bot de rinhas de galos para Discord! Este guia vai te mostrar tudo o que você precisa para começar sua jornada.

## Primeiros Passos

<Steps>
  <Step title="Descubra seu Galo">
    Use o comando `j!galo` para descobrir qual é o seu galo, seu **tipo** e sua **raridade**.

    Existem **6 raridades** de galos no jogo:

    | Raridade  | Cor da Borda |
    |-----------|-------------|
    | Comum     | Cinza       |
    | Raro      | Azul        |
    | Épico     | Roxo        |
    | Lendário  | Amarelo     |
    | Mítico    | Branco      |
    | Divino    | Magenta     |
  </Step>

  <Step title="Treine seu Galo">
    Use `j!train` para treinar seu galo. Cada treino concede **XP** e **Gold** caso você vença — ambos são essenciais para o crescimento do seu galo.

    <Info>
      Você pode vencer até **280 treinos por dia**.
    </Info>
  </Step>

  <Step title="Explore os Sistemas">
    Durante os treinos você pode ganhar **Chaves de Raid** e completar **Missões**. Confira suas missões com `j!mission` e saiba mais sobre raids com `j!raid`.
  </Step>
</Steps>

## Comandos Essenciais
