# Shards

As Shards são **fragmentos de raridade** usados para duas finalidades principais: **evoluir a raridade de galos** e **alimentar o sistema de Ovo**.

<img src="/imagens/upgradegalo.png" alt="Evoluindo a raridade de um galo com Shards" />

## Raridades de Shards

| Shard     | Evolui o galo para... |
|-----------|----------------------|
| Rara      | Raro                 |
| Épica     | Épico                |
| Lendária  | Lendário             |
| Mítica    | Mítico               |
| Divina    | Divino               |

<Info>
  Cada Shard só pode ser usada para evoluir para **sua própria raridade**. Você não pula etapas!
</Info>

---

## Como Evoluir a Raridade de um Galo

Para aumentar a raridade de um galo, você precisa de **9 Shards da raridade destino**. O processo é feito passo a passo:

```
Galo Comum   + 9 Shards Raras     → Galo Raro
Galo Raro    + 9 Shards Épicas    → Galo Épico
Galo Épico   + 9 Shards Lendárias → Galo Lendário
Galo Lendário + 9 Shards Míticas  → Galo Mítico
Galo Mítico  + 9 Shards Divinas   → Galo Divino
```

Use o comando `asura upgradegalo` para realizar a evolução.

---

## Usando Shards no Sistema de Ovo

As Shards também funcionam como **XP para o Ovo** — quanto maior a raridade da Shard, mais XP ela fornece.

Use `j!egg feed` para alimentar seu ovo com Shards. Veja mais detalhes na página de [Ovos](/pt/ovos).

## Como Conseguir Shards

<CardGroup cols={2}>
  <Card title="Raids" icon="shield">

  </Card>
  <Card title="Shop" icon="store">

  </Card>
  <Card title="Arena Shop" icon="sword">

  </Card>
  <Card title="Expedição" icon="compass">
    Deixar seu galo em expedição tem **chance de trazer Shards** de volta.
  </Card>
</CardGroup>

## Comércio

As Shards podem ser **vendidas**, **trocadas via Trade** ou **listadas na Store** do servidor — são itens valiosos para outros jogadores que querem evoluir suas raridades.
