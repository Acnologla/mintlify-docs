# Galinheiro

O Galinheiro é um sistema de **progressão passiva** onde seus galos evoluem automaticamente ao longo do tempo, sem precisar de rinhas ou treinos ativos.

## Como Funciona

- A cada ciclo de tempo, os galos colocados no galinheiro recebem recompensas automáticas
- Os bônus são concedidos **individualmente** para cada galo alocado
- Para remover um galo do galinheiro, basta selecioná-lo novamente no menu

---

## Recompensas por Ciclo

A cada ciclo, cada galo no galinheiro recebe:

- **XP**
- **UserXP**
- **XP para o Passe de Batalha**
- **Pequena chance de Chaves para Raids**

<Info>
  O Galinheiro é uma ótima fonte de progressão passiva — ideal para manter o avanço mesmo sem jogar ativamente.
</Info>

---

## Custo

Para cada galo colocado no galinheiro, **+15 treinos são adicionados ao seu limite diário** de treinos. Esse é o mecanismo de equilíbrio do sistema.

<Warning>
  Monitore seu limite de treinos! Usar muitos slots no galinheiro pode reduzir sua capacidade de treinar manualmente durante o dia.
</Warning>
