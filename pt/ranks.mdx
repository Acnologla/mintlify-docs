## Ranking por Galo Específico

Você pode ver o ranking de resets de um **galo específico**:

```
j!rank user level [nome do galo]
```

**Exemplo:**
```
j!rank user level black
```

Isso mostra quem tem mais resets no galo "Black" do servidor.

---

## Todos os Comandos de Ranking

| Comando                           | Função                                       |
|-----------------------------------|----------------------------------------------|
| `j!rank user level`               | Galos mais fortes                            |
| `j!rank user money`               | Jogadores com mais Gold                      |
| `j!rank user wins`                | Jogadores com mais vitórias                  |
| `j!rank user players`             | Maiores progressores da Dungeon              |
| `j!rank user portal`              | Mais portais completados                     |
| `j!rank user asuracoins`          | Mais AsuraCoins armazenados                  |
| `j!rank user pets`                | Pets de maior nível                          |
| `j!rank user level [nome]`        | Resets de um galo específico                 |
| `j!ranked rank`                   | Ranking geral de Elo da Ranqueada            |
